simulator.mason

MasonSwarm

class MasonSwarm extends SimState with Environment

Contiene el objeto de simulación de MASON. Además propociona el evento onStep, que se lanza para cada paso de la simulación. Si existen los métodos getX o setX provocarán que se muestre la propiedad X en el GUI del simulador.
